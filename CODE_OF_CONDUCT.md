# Code of Conduct

## Our Standard

We expect everyone in this project to be respectful, constructive, and professional.

Examples of positive behavior:

- Giving actionable, technical feedback without personal attacks.
- Respecting different experience levels and viewpoints.
- Assuming good intent and asking clarifying questions before escalating.
- Focusing on facts, reproducibility, and solutions.

Examples of unacceptable behavior:

- Harassment, discrimination, or hate speech.
- Personal insults, trolling, or repeated hostile behavior.
- Publishing private information without consent.
- Disruptive conduct that blocks productive collaboration.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and other community interactions.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards and may remove, edit, or reject contributions or interactions that violate this code.

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers privately via the contact path in `SECURITY.md`.
