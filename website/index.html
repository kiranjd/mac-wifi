FAQ

Short answers

What does MacWiFi actually test?

It checks things like throughput, latency, packet loss, router response, and DNS timing, then rolls that into a simple status.

Is this a speed test app?

Not really. Speed is part of it, but the point is to answer the more ordinary question of whether the connection seems usable.

Do I need an account?

No. You buy it once, activate it, and use it locally on your Mac.

Why does it ask for location permission?

macOS requires location access for apps that want to read SSID names during Wi-Fi scans. Without it, some network names may be hidden.
